# Pipeline 1 — Back/Forward 73-Split

A strategy pipeline that runs a backtest and an out-of-sample forward test on
a 70/30 time split of the available history, using `adrs`'s `AlphaReportV1`.

A full reference implementation is inlined at the bottom of this document —
see the [Reference implementation](#reference-implementation) section.

## Directory structure

Every alpha built with this pipeline lives under
`brain/strategy/YYYY-MM-DD_strategy_name/`:

```
alpha.py         # Alpha class (signal generation only)
backtest.py      # Runner: loads data, runs AlphaReportV1, writes report.parquet
plot.py          # Plot generator (copy verbatim from the reference implementation)
report.parquet   # AlphaReportV1 output (generated by backtest.py)
summary.md       # Frontmatter + strategy writeup
assets/          # Plots (generated by plot.py)
  equity_curve.png
  drawdown.png
  rolling_sharpe.png
  returns_distribution.png
  monthly_returns.png
```

## `alpha.py`

Only the `Alpha` subclass — no data loading, no plotting, no backtest logic.

- Subclass `Alpha` from `adrs`.
- Define `data_infos` and `data_processor` in `__init__`.
- Implement `next()` returning a DataFrame with a `signal` column.

## `backtest.py`

Mirrors the reference exactly. Only these parts change between strategies:

- Alpha import + constructor parameters
- `start_time`, `end_time`
- `Evaluator` asset configuration (topic, columns)
- `AlphaReportV1.compute` parameters (`fees`, `price_shift`, `base_asset`)

Requirements:

- `DataLoader` with `data_dir = os.environ["NYROBRAIN_DATA_DIR"]` (see `docs/conventions/storage-paths.md`).
- `report.write_parquet(Path(__file__).parent / "report.parquet")` with
  columns `["start_time", "price", "signal", "pnl"]`.
- Runnable via `uv run python backtest.py` from the strategy directory.

## `summary.md`

YAML frontmatter + writeup.

### Frontmatter

```yaml
---
name: Human-readable strategy name
status: failed | pass | keep-in-review
author: Your name
revision: 1
created_at: 2026-04-16T00:00:00Z
last_updated: 2026-04-16T00:00:00Z
---
```

- `name` — short title (dashboard falls back to folder name if omitted).
- `status` — **must be exactly one of these three strings, no alternatives**:
  - `pass` — genuine edge, approved for further development or live use
  - `failed` — structurally unprofitable or invalidated; archived
  - `keep-in-review` — partial promise or inconclusive; needs more work
  
  Do NOT use `success`, `fail`, `review`, or any other variant — only the exact strings above are recognised by the dashboard filter.
- `revision` — integer; increment on each meaningful update.
- `created_at` / `last_updated` — ISO 8601 datetime.

`status`, `created_at`, `last_updated` are required for dashboard sort/filter.

## Reference implementation

Full working example: `BTC Price Spread Z-Score` (pipeline 1,
back/forward 73-split). The four text files below — `alpha.py`,
`backtest.py`, `summary.md`, `plot.py` — are the complete source and all

### `alpha.py`

```python
import polars as pl
from typing import override

from adrs import Alpha
from adrs.data import DataInfo, DataColumn, DataProcessor


class PriceSpreadZScore(Alpha):
    def __init__(self, window: int, entry: float, exit: float) -> None:
        super().__init__(
            id="price_spread_zscore",
            data_infos=[
                DataInfo(
                    topic="binance-spot|candle?symbol=BTCUSDT&interval=1h",
                    columns=[DataColumn(src="close", dst="close_binance")],
                    lookback_size=window + 100,
                ),
                DataInfo(
                    topic="bybit-spot|candle?symbol=BTCUSDT&interval=1h",
                    columns=[DataColumn(src="close", dst="close_bybit")],
                    lookback_size=window + 100,
                ),
            ],
            data_processor=DataProcessor(),
        )
        self.window = window
        self.entry = entry
        self.exit = exit

    @override
    def next(self, data_df: pl.DataFrame) -> pl.DataFrame:
        spread = pl.col("close_bybit") - pl.col("close_binance")
        zscore = (spread - spread.rolling_mean(self.window)) / spread.rolling_std(
            self.window, ddof=1
        )
        df = data_df.with_columns(zscore.alias("zscore")).filter(
            pl.col("zscore").is_finite()
        )
        df = df.with_columns(
            pl.when(pl.col("zscore") >= self.entry)
            .then(1)
            .when(pl.col("zscore") <= self.exit)
            .then(0)
            .otherwise(None)
            .forward_fill()
            .fill_null(strategy="zero")
            .alias("signal")
        )
        return df
```

### `backtest.py`

```python
import os
import asyncio
from pathlib import Path
from datetime import datetime
from adrs.data import DataInfo, DataColumn, DataLoader, make_datamap
from adrs.utils import backforward_split
from adrs.performance import Evaluator
from adrs.logging import setup_logger
from adrs.report import AlphaReportV1

from alpha import PriceSpreadZScore

def _find_project_root(marker: str = "pyproject.toml") -> Path:
    p = Path(__file__).resolve()
    for parent in [p, *p.parents]:
        if (parent / marker).exists():
            return parent
    raise RuntimeError(f"{marker} not found above {__file__}")


PROJECT_ROOT = _find_project_root()


async def main():
    setup_logger()

    start_time = datetime.fromisoformat("2022-05-01T00:00:00Z")
    end_time = datetime.fromisoformat("2026-01-01T00:00:00Z")

    dataloader = DataLoader(
        data_dir=os.environ["NYROBRAIN_DATA_DIR"],
        credentials={"cybotrade_api_key": os.environ["USER_ID"]},
        cybotrade_api_url=os.environ["DATASOURCE_PROXY_URL"],
    )
    evaluator = Evaluator(
        assets={
            "BTC": DataInfo(
                topic="bybit-linear|candle?symbol=BTCUSDT&interval=1m",
                columns=[DataColumn(src="close", dst="price")],
                lookback_size=0,
            )
        }
    )
    alpha = PriceSpreadZScore(window=40, entry=0.825, exit=-0.825)

    datamap = await make_datamap(
        dataloader=dataloader,
        start_time=start_time,
        end_time=end_time,
        data_infos=alpha.data_infos,
        evaluator=evaluator,
    )

    data_df = alpha.data_processor.process(datamap)

    B_start, B_end, F_start, F_end = backforward_split(
        start_time=start_time, end_time=end_time, size=(0.7, 0.3)
    )
    report = AlphaReportV1.compute(
        alpha,
        B_start,
        B_end,
        F_start,
        F_end,
        evaluator=evaluator,
        base_asset="BTC",
        datamap=datamap,
        data_df=data_df,
        fees=0.035,
        price_shift=10,
    )
    print("backtest", report.back.performance)
    print("forward test", report.forward.performance)
    REPORT_COLS = ["start_time", "price", "signal", "pnl"]
    report.back.performance_df = report.back.performance_df.select(REPORT_COLS)
    report.forward.performance_df = report.forward.performance_df.select(REPORT_COLS)
    report.write_parquet(str(Path(__file__).parent / "report.parquet"))


if __name__ == "__main__":
    asyncio.run(main())
```

````markdown
# Price Spread Z-Score

## Alpha Hypothesis

Mean-reversion on the BTC spot price spread between Bybit and Binance. When the spread's z-score exceeds a threshold, the strategy assumes convergence will follow and takes a long position on the underlying asset (BTCUSDT perpetual). It exits when the z-score drops below the opposite threshold.

The implied counterparty: retail flow or latency-driven mispricing between the two venues creating temporary dislocations in spot price.

## Parameters

| Parameter | Value |
|-----------|-------|
| Window    | 40 (hours) |
| Entry     | 0.825 (z-score) |
| Exit      | -0.825 (z-score) |

## Data

- **Signal**: `binance-spot|candle?symbol=BTCUSDT&interval=1h`, `bybit-spot|candle?symbol=BTCUSDT&interval=1h`
- **Evaluation**: `bybit-linear|candle?symbol=BTCUSDT&interval=1m`
- **Period**: 2022-05-01 to 2026-01-01 (70/30 back/forward split)
- **Fees**: 3.5 bps per trade, 10-minute price shift

## Visualizations

### Equity Curve

![Equity Curve](assets/equity_curve.png)

The backtest period (left of the dashed line) shows a near-monotonic decline to -100%. The forward period oscillates around zero but ends slightly negative at -5.44%.

### Drawdown

![Drawdown](assets/drawdown.png)

The strategy enters a deep drawdown in mid-2022 and never fully recovers during the backtest period. The forward period adds a secondary drawdown of -25.6% before a partial recovery into year-end 2025.

### Rolling Sharpe Ratio

![Rolling Sharpe](assets/rolling_sharpe.png)

The 180d rolling Sharpe (blue) spends the vast majority of the sample below zero. Brief positive windows (late 2022, mid-2025) never sustain — the 30d Sharpe (orange) is noisy and mean-reverting around zero, confirming no persistent edge.

### PnL Distribution

![Returns Distribution](assets/returns_distribution.png)

Near-symmetric distribution centered just below zero (mean: -0.000011, median: -0.000012). The slight negative skew and fat left tail (losses extending to -8%) vs. right tail (+6%) show the strategy gives back more on its losers than it captures on its winners.

### Monthly Returns

![Monthly Returns](assets/monthly_returns.png)

The heatmap confirms there is no consistent seasonal edge. Red months dominate across all years — notably Jun 2022 (-39%), Sep 2024 (-19.4%), and Jan 2025 (-11.9%). The few green months are scattered with no repeatable pattern, reinforcing that the strategy has no structural alpha to exploit.

## Evaluation

**Verdict: Not viable.**

The strategy loses money in both the backtest and forward periods. While the win rate is marginally above 50% in both periods, the losses on losing trades far exceed the gains on winning trades — the strategy bleeds through adverse selection and transaction costs.

**Backtest period** is catastrophic: a -97.87% total return with a max drawdown of -113% that spans nearly the entire sample. The equity curve shows a near-monotonic decline from mid-2022 onward with no sustained recovery. This is not a strategy experiencing a bad regime — it is structurally unprofitable.

**Forward period** is less severe but still negative. The equity curve oscillates around zero before settling at -5.44%. The lower magnitude likely reflects a calmer spread environment rather than any alpha, since the strategy's core logic hasn't changed.

**Why this fails**: The Bybit-Binance BTC spot spread is extremely tight and well-arbitraged. Any mean-reverting signal in the spread is too small and too fast to capture at hourly resolution with 3.5 bps of fees. The strategy is effectively trading noise — the 52% win rate confirms near-random entry quality. The z-score threshold (0.825) is not extreme enough to isolate genuine dislocations from normal microstructure noise.

**What would need to change**: This class of strategy requires either (a) sub-second execution on a genuinely dislocated pair, or (b) a spread with a much stronger economic reason to mean-revert (e.g., funding rate vs. spot, or cross-exchange futures basis). The current setup has neither.
````

### `plot.py`

```python
"""
Generate visualization plots from an AlphaReportV1 parquet file for the
back/forward 73-split pipeline.

Copy this file verbatim into the strategy directory — it reads
`report.parquet` from its own directory and writes to `./assets/`.

Usage (from anywhere):
    uv run python path/to/strategy/plot.py
"""

from pathlib import Path

import io
import numpy as np
import polars as pl
import matplotlib

matplotlib.use("Agg")
import matplotlib.pyplot as plt
import matplotlib.dates as mdates


STRATEGY_DIR = Path(__file__).resolve().parent


def load_report(strategy_dir: Path):
    # New backtests write `report.parquet`; older strategies on disk may
    # still have `report.parquet.png` from a prior fs-encoding workaround.
    report_path = strategy_dir / "report.parquet"
    if not report_path.exists():
        legacy = strategy_dir / "report.parquet.png"
        if legacy.exists():
            report_path = legacy
    df = pl.read_parquet(report_path)
    row = df.row(0, named=True)
    back_df = pl.read_parquet(io.BytesIO(row["back_pdf_buf"]))
    forward_df = pl.read_parquet(io.BytesIO(row["forward_pdf_buf"]))
    full_df = pl.concat([back_df, forward_df])
    split_time = back_df["start_time"][-1]
    return full_df, split_time


def plot_equity_curve(full_df: pl.DataFrame, split_time, out: Path):
    times = full_df["start_time"].to_list()
    equity = full_df["pnl"].cum_sum().to_numpy()

    fig, ax = plt.subplots(figsize=(14, 5))
    ax.plot(times, equity, linewidth=0.6, color="#c0392b")
    ax.axvline(
        x=split_time,
        color="#2c3e50",
        linestyle="--",
        linewidth=1,
        label="Back/Forward Split",
    )
    ax.axhline(y=0, color="gray", linestyle="-", linewidth=0.4, alpha=0.5)
    ax.set_title("Equity Curve (Arithmetic Returns)")
    ax.set_ylabel("Cumulative Return")
    ax.legend()
    ax.grid(True, alpha=0.2)
    ax.xaxis.set_major_formatter(mdates.DateFormatter("%Y-%m"))
    ax.xaxis.set_major_locator(mdates.MonthLocator(interval=4))
    fig.autofmt_xdate()
    plt.tight_layout()
    plt.savefig(out, dpi=150)
    plt.close()


def plot_rolling_sharpe(full_df: pl.DataFrame, split_time, out: Path):
    times = full_df["start_time"].to_list()
    pnl = full_df["pnl"]

    roll_30d = 720  # 30 days in hours
    roll_180d = 4320  # 180 days in hours
    ann_factor = np.sqrt(8760)

    sharpe_30d = (
        (pnl.rolling_mean(roll_30d) / pnl.rolling_std(roll_30d)) * ann_factor
    ).to_numpy()
    sharpe_180d = (
        (pnl.rolling_mean(roll_180d) / pnl.rolling_std(roll_180d)) * ann_factor
    ).to_numpy()

    fig, ax = plt.subplots(figsize=(14, 5))
    ax.plot(
        times,
        sharpe_30d,
        linewidth=0.7,
        color="#e67e22",
        alpha=0.8,
        label="30d Rolling Sharpe",
    )
    ax.plot(
        times, sharpe_180d, linewidth=1.2, color="#2980b9", label="180d Rolling Sharpe"
    )
    ax.axvline(
        x=split_time,
        color="#2c3e50",
        linestyle="--",
        linewidth=1,
        label="Back/Forward Split",
    )
    ax.axhline(y=0, color="gray", linestyle="-", linewidth=0.4, alpha=0.5)
    ax.set_title("Rolling Sharpe Ratio (Annualized)")
    ax.set_ylabel("Sharpe Ratio")
    ax.set_ylim(-5, 5)
    ax.legend()
    ax.grid(True, alpha=0.2)
    ax.xaxis.set_major_formatter(mdates.DateFormatter("%Y-%m"))
    ax.xaxis.set_major_locator(mdates.MonthLocator(interval=4))
    fig.autofmt_xdate()
    plt.tight_layout()
    plt.savefig(out, dpi=150)
    plt.close()


def plot_returns_distribution(full_df: pl.DataFrame, out: Path):
    pnl = full_df["pnl"].to_numpy()
    prev_signal = full_df["signal"].shift(1).fill_null(0).to_numpy()
    active_pnl = pnl[prev_signal == 1]
    active_pnl = active_pnl[np.isfinite(active_pnl)]

    fig, ax = plt.subplots(figsize=(14, 5))
    ax.hist(active_pnl, bins=200, color="#2c3e50", alpha=0.7, edgecolor="none")
    ax.axvline(
        x=np.mean(active_pnl),
        color="#c0392b",
        linestyle="--",
        linewidth=1.2,
        label=f"Mean: {np.mean(active_pnl):.6f}",
    )
    ax.axvline(
        x=np.median(active_pnl),
        color="#2980b9",
        linestyle="--",
        linewidth=1.2,
        label=f"Median: {np.median(active_pnl):.6f}",
    )
    ax.set_title("PnL Distribution (Active Positions Only)")
    ax.set_xlabel("Hourly PnL")
    ax.set_ylabel("Frequency")
    ax.legend()
    ax.grid(True, alpha=0.2)
    plt.tight_layout()
    plt.savefig(out, dpi=150)
    plt.close()


def plot_monthly_returns(full_df: pl.DataFrame, out: Path):
    monthly = (
        full_df.with_columns(
            pl.col("start_time").dt.year().alias("year"),
            pl.col("start_time").dt.month().alias("month"),
        )
        .group_by("year", "month")
        .agg(pl.col("pnl").sum().alias("monthly_return"))
        .sort("year", "month")
    )

    years = sorted(monthly["year"].unique().to_list())
    month_labels = [
        "Jan",
        "Feb",
        "Mar",
        "Apr",
        "May",
        "Jun",
        "Jul",
        "Aug",
        "Sep",
        "Oct",
        "Nov",
        "Dec",
    ]

    matrix = np.full((len(years), 12), np.nan)
    for row in monthly.iter_rows(named=True):
        yi = years.index(row["year"])
        mi = row["month"] - 1
        matrix[yi, mi] = row["monthly_return"] * 100

    fig, ax = plt.subplots(figsize=(14, 4))
    vmax = np.nanmax(np.abs(matrix))
    im = ax.imshow(matrix, cmap="RdYlGn", aspect="auto", vmin=-vmax, vmax=vmax)

    ax.set_xticks(range(12))
    ax.set_xticklabels(month_labels)
    ax.set_yticks(range(len(years)))
    ax.set_yticklabels(years)

    for i in range(len(years)):
        for j in range(12):
            val = matrix[i, j]
            if not np.isnan(val):
                color = "white" if abs(val) > vmax * 0.6 else "black"
                ax.text(
                    j,
                    i,
                    f"{val:.1f}%",
                    ha="center",
                    va="center",
                    fontsize=9,
                    color=color,
                    fontweight="bold",
                )

    ax.set_title("Monthly Returns (%)")
    cbar = fig.colorbar(im, ax=ax, shrink=0.8, pad=0.02)
    cbar.set_label("Return (%)")
    plt.tight_layout()
    plt.savefig(out, dpi=150)
    plt.close()


def plot_drawdown(full_df: pl.DataFrame, split_time, out: Path):
    times = full_df["start_time"].to_list()
    pnl = full_df["pnl"].to_numpy()
    cumulative = np.cumsum(pnl)
    running_max = np.maximum.accumulate(cumulative)
    drawdown = cumulative - running_max

    fig, ax = plt.subplots(figsize=(14, 5))
    ax.fill_between(times, drawdown, 0, color="#c0392b", alpha=0.4)
    ax.plot(times, drawdown, linewidth=0.5, color="#c0392b")
    ax.axvline(
        x=split_time,
        color="#2c3e50",
        linestyle="--",
        linewidth=1,
        label="Back/Forward Split",
    )
    ax.set_title("Drawdown")
    ax.set_ylabel("Drawdown (Arithmetic)")
    ax.legend()
    ax.grid(True, alpha=0.2)
    ax.xaxis.set_major_formatter(mdates.DateFormatter("%Y-%m"))
    ax.xaxis.set_major_locator(mdates.MonthLocator(interval=4))
    fig.autofmt_xdate()
    plt.tight_layout()
    plt.savefig(out, dpi=150)
    plt.close()


def main():
    assets_dir = STRATEGY_DIR / "assets"
    assets_dir.mkdir(exist_ok=True)

    full_df, split_time = load_report(STRATEGY_DIR)

    plot_equity_curve(full_df, split_time, assets_dir / "equity_curve.png")
    print("  equity_curve.png")

    plot_rolling_sharpe(full_df, split_time, assets_dir / "rolling_sharpe.png")
    print("  rolling_sharpe.png")

    plot_returns_distribution(full_df, assets_dir / "returns_distribution.png")
    print("  returns_distribution.png")

    plot_monthly_returns(full_df, assets_dir / "monthly_returns.png")
    print("  monthly_returns.png")

    plot_drawdown(full_df, split_time, assets_dir / "drawdown.png")
    print("  drawdown.png")

    print(f"Done. Plots saved to {assets_dir}/")


if __name__ == "__main__":
    main()
```
