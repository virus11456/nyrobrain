# Pipeline 2 — Portfolio Backtest

## Prerequisites

This pipeline requires `adrs >= 1.2.1`. Check the installed version before writing any code:

```bash
uv run python -c "import adrs; print(adrs.__version__)"
```

If the version is below `1.2.1`, update it first:

```bash
uv add "adrs>=1.2.1"
```

Do not proceed with code generation until the version requirement is confirmed.

---

A portfolio pipeline that combines multiple alphas into a single portfolio
and backtests it with the `adrs.portfolio` API (`adrs >= 1.2.1`):
alphas grouped into `Asset` objects, combined into a `Portfolio`, and
backtested via `Portfolio.backtest()` which returns
`(PortfolioPerformance, DataFrame[PortfolioPerformanceDF])`.

This pipeline takes strategies from `brain/strategy/` (already-written `Alpha` classes)
and evaluates them together as a portfolio. The default allocator is `mean_alpha_allocator`
(equal weight within each asset group) combined with `mean_asset_allocator` (equal weight
across asset groups). Use a custom `asset_allocator` when alphas are unevenly distributed
across assets and flat per-alpha weight is desired.

**Scope: backtest only.** This pipeline produces backtest code and results.
Live execution and order routing are out of scope — never generate them.

A full reference implementation is inlined at the bottom of this document —
see the [Reference implementation](#reference-implementation) section.

## Directory structure

Every portfolio built with this pipeline lives under
`brain/portfolio/YYYY-MM-DD_portfolio_name/`:

```
portfolio.py          # setup_portfolio(): alpha registry, metadata, signals, weights
backtest.py           # Runner: builds prices_df, runs Portfolio.backtest, writes parquets
plot.py               # Plot generator (copy verbatim from the reference implementation)
report.parquet        # Hourly aggregated portfolio pnl (generated by backtest.py)
attribution.parquet   # Hourly per-alpha pnl (generated by backtest.py)
signals.parquet       # Hourly per-alpha signals (generated by backtest.py)
metadata.parquet      # Alpha metadata: custom_id, base_asset (generated by backtest.py)
prices.parquet        # Hourly asset prices (generated by backtest.py)
summary.md            # Frontmatter + portfolio writeup
assets/               # Plots (generated by plot.py)
  equity_curve.png
  drawdown.png
  rolling_sharpe.png
  returns_distribution.png
  monthly_returns.png
  alpha_attribution.png    # Monthly per-alpha PnL heatmap
  correlation_matrix.png   # Full-sample pairwise signal correlation heatmap
  correlation_pairs.png    # Top 25 alpha pairs by |ρ|, horizontal bar chart
  exposure.png             # Net portfolio exposure over time (per asset + total)
  asset_drawdown.png       # Per-asset underwater curves
  market_beta.png          # Portfolio vs market daily returns scatter (OLS)
  capm_decomposition.png   # Portfolio cumulative return decomposed into α + β×market
```

A portfolio is documented exactly like a strategy: every portfolio ships a
`summary.md` with all twelve plots — see [`summary.md`](#summarymd) below.

## Alpha sources — no forced module format

The portfolio accepts **any `adrs.Alpha` subclass** — whatever implements
`next()` returning a DataFrame with a `signal` column in [-1, 1]. Do not
require alphas to live in specially named files; in particular, 
strategies stay where they are (`brain/strategy/YYYY-MM-DD_name/alpha.py`)
and are loaded **by file path** with the `load_alpha_class` helper (see
reference implementation), since dated directory names are not importable
module names:

```python
MA50Crossover = load_alpha_class(
    PROJECT_ROOT / "brain/strategy/2026-05-28_btc_ma50_crossover/alpha.py",
    "MA50Crossover",
)
```

Regular importable modules work too — both kinds mix freely in one portfolio.
The only identity that matters downstream is the alpha's `id`, which keys
its column in `signal_df` and its row in `metadata_df`. Every `id` in the
portfolio must be unique.

## Datasource — proxy only

All data goes through the nyrobrain datasource proxy. **Never** read a raw
cybotrade API key from `credentials.json` or any other file — the only
credentials are these two environment variables (already present in the
nyrobrain environment):

| Variable | Purpose |
| -------- | ------- |
| `USER_ID` | passed as the `cybotrade_api_key` credential |

```python
dataloader = DataLoader(
```

## `portfolio.py`

Exports a single `async setup_portfolio(start_time=<default>, end_time=<default>, live=False)`
returning `(portfolio, alphas, evaluator, datamap)`. `alphas` is the flat list of alpha instances from the registry (the same list flat-mapped for `data_infos`); the live OMS runner needs it, and backtests ignore it.

**`start_time` / `end_time` (params with defaults):** expose both as parameters
with reproducible default dates, and thread them to BOTH `make_datamap(...)` and
`generate_signal_df(...)`. `backtest.py` passes both explicitly (intentional
backtest window). The live OMS runner (`oms/run.py`) passes
`end_time=datetime.now(tz=utc)` so `signal_df` reaches current time —
**`PortfolioExecutor` rejects a `signal_df` whose last row is too old**
("signal_df is stale"). `end_time=now` only helps if the datasource actually has
rows up to ~now; a lagging feed or a stale `outdir` cache still yields an old
last row.

**`live` (default False):** `oms/run.py` sets `live=True`; pass it to
`generate_signal_df(..., forward_fill_to_end=live)`. That forward-fills each
alpha's last signal to `end_time`, so a **mixed-cadence** portfolio (e.g. a 1h
alpha + a 24h alpha) still produces a `signal_df` reaching ~now even though the
24h alpha's last bar is older. Keep it `False` for backtests so results are
unchanged. (adrs >= 1.2.8 also widens the staleness window to the finest
interval and tolerates a not-yet-closed slow-topic bar, so daily portfolios run
too.)

Only these parts change between portfolios:

- The alpha registry — a list of `(alpha_instance, base_asset)` pairs.
- `start_time` (and the default `end_time` when none is passed).
- The `Evaluator` asset map — one `DataInfo` per traded asset, 1m candles,
  `close → price`, `lookback_size=0`. Every `base_asset` named in the
  registry must have an evaluator entry.
- Per-alpha `fees` / `shift_backtest_candle_minute` in `metadata_df`
  (defaults: 0.035 fees, 70-minute shift).
- The `Portfolio` `id`.

The assembly sequence is fixed:

1. **`metadata_df`** — one row per alpha:
   `custom_id` (the alpha's `id`), `base_asset`,
   `shift_backtest_candle_minute`, `fees`.
2. **Datamap** — flat-map `data_infos` over every alpha in the registry
   (never hand-maintain the list), then `make_datamap(...)` with the proxy
   `DataLoader` and `evaluator_offset=timedelta()`.
3. **`signal_df`** — `generate_signal_df(alphas, metadata_df, evaluator,
   datamap, start_time, end_time)`. One `start_time` column plus one
   signal column per alpha `id`. Assert it is not `None`.
4. **`weight_df`** — `MeanWeightAllocator(signal_df, metadata_df).weights()`
   (equal weight per alpha; swap in another `WeightAllocator` for a
   different scheme).
5. **`Portfolio(id=..., signal_df=..., metadata_df=..., weight_df=...)`**.

## `backtest.py`

Mirrors the reference exactly. Builds the price matrix, runs the backtest,
and persists the pnl series for plotting:

- `prices_df` — one `start_time` column plus one 1m price column per
  evaluator asset, pulled straight from the datamap
  (`datamap.get(info)`), full-joined on `start_time`.
- `performance, performance_df = portfolio.backtest(prices_df)` — returns a
  tuple: `PortfolioPerformance` (sharpe/calmar/sortino, CAGR, annualized and
  total return, max-drawdown block, per-asset `TradePerformance`) and a
  `DataFrame[PortfolioPerformanceDF]` with columns `start_time, pnl, equity`
  at 1m resolution. Do **not** reimplement `build_performance_df` — the
  library returns it.
- Writes five artefacts next to itself:
  - `report.parquet` — hourly aggregated `["start_time", "pnl"]`
  - `attribution.parquet` — hourly per-alpha pnl, one column per alpha `id`
  - `signals.parquet` — hourly per-alpha signals (last value in each hour)
  - `metadata.parquet` — `["custom_id", "base_asset"]` from `metadata_df`
  - `prices.parquet` — hourly asset prices (`["start_time", "BTC", "ETH", ...]`), used by `market_beta.png` and `capm_decomposition.png`
- Runnable via `uv run python backtest.py` from the project root.

## `summary.md`

Same discipline as strategies: YAML frontmatter + writeup, all
twelve plots embedded.

### Frontmatter

```yaml
---
name: Human-readable portfolio name
status: failed | pass | keep-in-review
author: Your name
revision: 1
created_at: 2026-06-04T00:00:00Z
last_updated: 2026-06-04T00:00:00Z
---
```

`status` — **must be exactly one of these three strings, no alternatives**:
- `pass` — portfolio cleared evaluation, ready for further development or live use
- `failed` — structurally unprofitable or invalidated; archived
- `keep-in-review` — partial promise or inconclusive; needs more work

Do NOT use `success`, `fail`, `review`, or any other variant — only the exact strings above are recognised by the dashboard filter.

`created_at`, `last_updated` are required for dashboard sort/filter; increment `revision` on each meaningful update.

### Body

```
# {Portfolio Name}

## Composition
Table of alphas in the portfolio: id, source (strategy dir or local class),
base asset, key parameters, weight.

## Data
Evaluation topics, period, fees, price shift, allocator used.

## Backtest Results ({period})
Metrics table (see below).

## Visualizations
All 12 plots from assets/ with commentary.

## Evaluation
Verdict (viable / not viable / needs further investigation), which alphas
help or hurt the portfolio, and what to change next.
```

### Metrics table

| Metric | Source |
| ------ | ------ |
| Total Return (%) | `total_return` |
| Annualised Avg Return | `annualized_return` |
| CAGR (%) | `cagr` |
| Sharpe | `sharpe_ratio` |
| Sortino | `sortino_ratio` |
| Calmar | `calmar_ratio` |
| Market Beta (vs primary market, OLS daily) | `np.cov(port_ret, mkt_ret)[0,1] / np.var(mkt_ret)` on daily aggregated returns from `report.parquet` + `prices.parquet` |
| Pearson Correlation vs market (daily) | `np.corrcoef(port_ret, mkt_ret)[0,1]` |
| R² vs market (daily) | `corr ** 2` |
| Jensen's Alpha (daily) | OLS intercept: `mean(port_ret) - beta * mean(mkt_ret)` |
| Jensen's Alpha (annualized) | daily α × 365 |
| Information Ratio (IR) | `mean(active_ret) * 365 / (std(active_ret) * sqrt(365))` where `active_ret = port_ret - beta * mkt_ret` |
| Max Drawdown (%) | `max_drawdown` |
| MDD start/end/recovery | `max_drawdown_*_date` |
| Per-asset trades | `trades[asset]`: num_trades, win_rate, streaks, largest win/loss, avg/max holding time |

## Generating plots

Copy `plot.py` from the reference implementation verbatim into the
portfolio directory — it reads `report.parquet`, `attribution.parquet`,
`signals.parquet`, `metadata.parquet`, and `prices.parquet` from its own
directory and writes twelve PNGs to `./assets/`:

```
uv run python brain/portfolio/YYYY-MM-DD_portfolio_name/plot.py
```

The seven additional plots (`alpha_attribution.png`, `correlation_matrix.png`,
`correlation_pairs.png`, `exposure.png`, `asset_drawdown.png`, `market_beta.png`,
`capm_decomposition.png`) are generated only when their source parquets exist.
All are asset-agnostic: adding SOL, ETH, or any other asset to the portfolio
automatically produces the correct number of subplot rows, heatmap rows, exposure
panels, and beta series without any code changes. `market_beta.png` and
`capm_decomposition.png` use BTC as the primary market reference when present,
otherwise the first asset alphabetically. `correlation_pairs.png` always shows the top 25 pairs by |ρ|. When the
portfolio has ≤30 alphas, both `correlation_matrix.png` and
`correlation_pairs.png` are generated. When N > 30, the matrix is skipped
(cells become unreadably small) and only the pairs chart is produced.

## Checklist for adding an alpha to the portfolio

1. Alpha passes its evaluation (or at minimum runs standalone
   via `Alpha.backtest`).
2. Append `(instance, base_asset)` to the registry in `portfolio.py` —
   loading by file path if it lives in a dated strategy directory.
3. Confirm its `id` is unique within the portfolio.
4. Confirm the `base_asset` has an `Evaluator` entry.
5. Re-run `uv run python backtest.py` — portfolio boots, datamap resolves
   all topics, performance prints, `report.parquet` is rewritten.
6. Re-run `uv run python plot.py` and update `summary.md` (bump
   `revision`, refresh metrics tables and plot commentary).

## Reference implementation

Minimal two-alpha, two-asset portfolio: one strategy loaded by file
path (BTC) plus one locally defined alpha (ETH). Both files live in the
same portfolio directory under `brain/portfolio/`.

### `portfolio.py`

```python
import os
import importlib.util
from pathlib import Path
from datetime import datetime, timedelta

import polars as pl

from adrs import alpha
from adrs.portfolio import MeanWeightAllocator, Portfolio
from adrs.performance import Evaluator
from adrs.data import DataColumn, DataInfo, DataLoader, DataProcessor, make_datamap
from adrs.execution import generate_signal_df


def _find_project_root(marker: str = "pyproject.toml") -> Path:
    p = Path(__file__).resolve()
    for parent in [p, *p.parents]:
        if (parent / marker).exists():
            return parent
    raise RuntimeError(f"{marker} not found above {__file__}")


PROJECT_ROOT = _find_project_root()


def load_alpha_class(path: Path | str, class_name: str):
    """Load an Alpha class from a .py file that is not on the import path
    (e.g. brain/strategy/YYYY-MM-DD_name/alpha.py)."""
    path = Path(path)
    spec = importlib.util.spec_from_file_location(f"{path.stem}_{class_name}", path)
    assert spec is not None and spec.loader is not None
    mod = importlib.util.module_from_spec(spec)
    spec.loader.exec_module(mod)
    return getattr(mod, class_name)


def flat_map(f, xs):
    ys = []
    for x in xs:
        ys.extend(f(x))
    return ys


# A locally defined alpha — any adrs.Alpha subclass is accepted.
class EthSpreadZScore(Alpha):
    def __init__(self, window: int, entry: float) -> None:
        super().__init__(
            id="eth_spread_zscore",
            data_infos=[
                DataInfo(
                    topic="binance-spot|candle?symbol=ETHUSDT&interval=1h",
                    columns=[DataColumn(src="close", dst="close_venue_a")],
                    lookback_size=window + 100,
                ),
                DataInfo(
                    topic="bybit-spot|candle?symbol=ETHUSDT&interval=1h",
                    columns=[DataColumn(src="close", dst="close_venue_b")],
                    lookback_size=window + 100,
                ),
            ],
            data_processor=DataProcessor(),
        )
        self.window = window
        self.entry = entry

    def next(self, data_df: pl.DataFrame) -> pl.DataFrame:
        spread = pl.col("close_venue_b") - pl.col("close_venue_a")
        zscore = (spread - spread.rolling_mean(self.window)) / spread.rolling_std(
            self.window, ddof=1
        )
        df = data_df.with_columns(zscore.alias("zscore")).filter(
            pl.col("zscore").is_finite()
        )
        return df.with_columns(
            pl.when(pl.col("zscore") >= self.entry)
            .then(1)
            .when(pl.col("zscore") <= -self.entry)
            .then(0)
            .otherwise(None)
            .forward_fill()
            .fill_null(strategy="zero")
            .alias("signal")
        )


async def setup_portfolio(
    start_time: datetime = datetime.fromisoformat("2024-01-01T00:00:00Z"),
    end_time: datetime = datetime.fromisoformat("2026-01-01T00:00:00Z"),
    live: bool = False,
):
    # start_time / end_time are parameters with reproducible defaults. backtest.py
    # passes both explicitly; the live OMS runner (oms/run.py) passes end_time=now
    # so signal_df reaches current time (PortfolioExecutor rejects a stale last
    # row). Both are threaded to make_datamap + generate_signal_df below.
    #
    # `live` (set True by oms/run.py): forward-fill signals to end_time so a
    # mixed-cadence portfolio (e.g. 1h + 24h alphas) still reaches ~now. Keep
    # False for backtests so results are unchanged.

    # Strategy loaded by file path — not a module import.
    MA50Crossover = load_alpha_class(
        PROJECT_ROOT / "brain/strategy/2026-05-28_btc_ma50_crossover/alpha.py",
        "MA50Crossover",
    )

    # Alpha registry: (instance, base_asset). Any Alpha subclass, any source.
    registry: list[tuple[Alpha, str]] = [
        (MA50Crossover(window=50), "BTC"),
        (EthSpreadZScore(window=48, entry=1.5), "ETH"),
    ]
    alphas = [a for a, _ in registry]

    evaluator = Evaluator(
        assets={
            "BTC": DataInfo(
                topic="bybit-linear|candle?symbol=BTCUSDT&interval=1m",
                columns=[DataColumn(src="close", dst="price")],
                lookback_size=0,
            ),
            "ETH": DataInfo(
                topic="binance-spot|candle?symbol=ETHUSDT&interval=1m",
                columns=[DataColumn(src="close", dst="price")],
                lookback_size=0,
            ),
        }
    )

    metadata_df = pl.DataFrame(
        {
            "custom_id": [a.id for a in alphas],
            "base_asset": [asset for _, asset in registry],
            "shift_backtest_candle_minute": [70] * len(alphas),
            "fees": [0.035] * len(alphas),
        }
    )

    dataloader = DataLoader(
        data_dir=os.environ["NYROBRAIN_DATA_DIR"],
        credentials={"cybotrade_api_key": os.environ["USER_ID"]},
        cybotrade_api_url=os.environ["DATASOURCE_PROXY_URL"],
    )
    datamap = await make_datamap(
        dataloader=dataloader,
        start_time=start_time,
        end_time=end_time,
        data_infos=list(flat_map(lambda a: a.data_infos, alphas)),
        evaluator=evaluator,
        evaluator_offset=timedelta(),
    )

    signal_df = generate_signal_df(
        alphas=alphas,
        metadata_df=metadata_df,
        evaluator=evaluator,
        datamap=datamap,
        start_time=start_time,
        end_time=end_time,
        forward_fill_to_end=live,  # live: carry last signal to ~now (off for backtests)
    )
    assert signal_df is not None

    weight_df = MeanWeightAllocator(signal_df, metadata_df).weights()

    portfolio = Portfolio(
        id="EXAMPLE_PORTFOLIO",
        signal_df=signal_df,
        metadata_df=metadata_df,
        weight_df=weight_df,
    )
    return portfolio, alphas, evaluator, datamap
```

### `backtest.py`

```python
import asyncio
from datetime import datetime
from pathlib import Path

import polars as pl

from portfolio import setup_portfolio


async def main():
    # Backtest window: pass start_time + end_time explicitly (don't rely on the
    # defaults) so the backtest range is intentional and reproducible.
    portfolio, _alphas, evaluator, datamap = await setup_portfolio(
        start_time=datetime.fromisoformat("2024-01-01T00:00:00Z"),
        end_time=datetime.fromisoformat("2026-01-01T00:00:00Z"),
    )

    # One 1m price column per evaluator asset, full-joined on start_time.
    prices_df: pl.DataFrame | None = None
    for asset, info in evaluator.assets.items():
        df = datamap.get(info).select("start_time", pl.col("price").alias(asset))
        prices_df = (
            df
            if prices_df is None
            else prices_df.join(
                df, on="start_time", how="full", coalesce=True
            ).sort("start_time")
        )
    assert prices_df is not None

    # backtest() returns (PortfolioPerformance, DataFrame[PortfolioPerformanceDF]).
    # The df has start_time, pnl, equity at 1m resolution — no need to rebuild it.
    performance, performance_df = portfolio.backtest(prices_df)
    print(performance)

    out = Path(__file__).parent

    # Hourly aggregated pnl → report.parquet
    hourly = (
        performance_df.sort("start_time")
        .group_by_dynamic("start_time", every="1h")
        .agg(pl.col("pnl").sum())
    )
    hourly.write_parquet(out / "report.parquet")

    # Per-alpha hourly pnl → attribution.parquet  (see build_attribution_df in the
    # full reference portfolio — replicates the library's per-alpha PnL formula)
    attribution_df = build_attribution_df(portfolio, prices_df)
    attribution_df.write_parquet(out / "attribution.parquet")

    # Hourly per-alpha signals → signals.parquet
    signal_cols = [c for c in portfolio.signal_df.columns if c != "start_time"]
    signals_hourly = (
        portfolio.signal_df.sort("start_time")
        .group_by_dynamic("start_time", every="1h")
        .agg([pl.col(c).last() for c in signal_cols])
    )
    signals_hourly.write_parquet(out / "signals.parquet")

    # Alpha metadata (id → base_asset) → metadata.parquet
    portfolio.metadata_df.select("custom_id", "base_asset").write_parquet(
        out / "metadata.parquet"
    )

    # Hourly prices per asset → prices.parquet (for market_beta / capm_decomposition plots)
    asset_cols = [c for c in prices_df.columns if c != "start_time"]
    prices_hourly = (
        prices_df.sort("start_time")
        .group_by_dynamic("start_time", every="1h")
        .agg([pl.col(a).last() for a in asset_cols])
    )
    prices_hourly.write_parquet(out / "prices.parquet")

    print("report.parquet, attribution.parquet, signals.parquet, metadata.parquet, prices.parquet written.")


if __name__ == "__main__":
    asyncio.run(main())
```

### `plot.py`

```python
"""
Generate visualization plots from a portfolio backtest report for the
portfolio-backtest pipeline.

Reads from the portfolio directory:
  report.parquet       — hourly aggregated pnl (start_time, pnl)
  attribution.parquet  — hourly per-alpha pnl (start_time, {alpha_id}, ...)
  signals.parquet      — hourly per-alpha signals (start_time, {alpha_id}, ...)
  metadata.parquet     — alpha metadata (custom_id, base_asset)
  prices.parquet       — hourly asset prices (start_time, BTC, ETH, ...)

Writes to ./assets/:
  equity_curve.png, drawdown.png, rolling_sharpe.png,
  returns_distribution.png, monthly_returns.png,
  alpha_attribution.png, correlation_matrix.png, correlation_pairs.png,
  exposure.png, asset_drawdown.png,
  market_beta.png, capm_decomposition.png

Usage:
    uv run python path/to/portfolio/plot.py
"""

from pathlib import Path

import numpy as np
from numpy.lib.stride_tricks import sliding_window_view
import polars as pl
import matplotlib

matplotlib.use("Agg")
import matplotlib.pyplot as plt
import matplotlib.dates as mdates
from matplotlib.colors import TwoSlopeNorm
from matplotlib.collections import LineCollection


PORTFOLIO_DIR = Path(__file__).resolve().parent


def load_report(portfolio_dir: Path) -> pl.DataFrame:
    return pl.read_parquet(portfolio_dir / "report.parquet").sort("start_time")


def plot_equity_curve(df: pl.DataFrame, out: Path):
    times = df["start_time"].to_list()
    equity = df["pnl"].cum_sum().to_numpy()

    fig, ax = plt.subplots(figsize=(14, 5))
    ax.plot(times, equity, linewidth=0.6, color="#c0392b")
    ax.axhline(y=0, color="gray", linestyle="-", linewidth=0.4, alpha=0.5)
    ax.set_title("Portfolio Equity Curve (Arithmetic Returns)")
    ax.set_ylabel("Cumulative Return")
    ax.grid(True, alpha=0.2)
    ax.xaxis.set_major_formatter(mdates.DateFormatter("%Y-%m"))
    ax.xaxis.set_major_locator(mdates.MonthLocator(interval=4))
    fig.autofmt_xdate()
    plt.tight_layout()
    plt.savefig(out, dpi=150)
    plt.close()


def plot_rolling_sharpe(df: pl.DataFrame, out: Path):
    times = df["start_time"].to_list()
    pnl = df["pnl"]

    roll_30d = 720
    roll_180d = 4320
    ann_factor = np.sqrt(8760)

    sharpe_30d = (
        (pnl.rolling_mean(roll_30d) / pnl.rolling_std(roll_30d)) * ann_factor
    ).to_numpy()
    sharpe_180d = (
        (pnl.rolling_mean(roll_180d) / pnl.rolling_std(roll_180d)) * ann_factor
    ).to_numpy()

    fig, ax = plt.subplots(figsize=(14, 5))
    ax.plot(times, sharpe_30d, linewidth=0.7, color="#e67e22", alpha=0.8, label="30d Rolling Sharpe")

def _build_corr_matrix(portfolio_dir: Path):
    """Shared: returns (corr, alpha_sorted, asset_map, labels, assets)."""
    signals = pl.read_parquet(portfolio_dir / "signals.parquet").sort("start_time")
    meta = pl.read_parquet(portfolio_dir / "metadata.parquet")
    asset_map = dict(meta.iter_rows())
    alpha_ids = [c for c in signals.columns if c != "start_time"]

    daily = (
        signals.group_by_dynamic("start_time", every="1d")
        .agg([pl.col(a).last() for a in alpha_ids])
        .sort("start_time")
    )
    mat = daily.select(alpha_ids).to_numpy().astype(float)

    assets = sorted(set(asset_map.values()))
    order = sorted(range(len(alpha_ids)),
                   key=lambda i: (asset_map.get(alpha_ids[i], "ZZZ"), alpha_ids[i]))
    alpha_sorted = [alpha_ids[i] for i in order]
    mat_sorted = mat[:, order]
    corr = np.corrcoef(mat_sorted.T)

    def short(aid: str) -> str:
        return aid.replace("_alpha", "").replace("_btc", "").replace("_zscore", "_z")

    labels = [short(a) for a in alpha_sorted]
    return corr, alpha_sorted, asset_map, labels, assets


def plot_rolling_correlation(portfolio_dir: Path, out: Path):
    """Full-sample pairwise signal correlation matrix heatmap (alphas × alphas).

    NaN cells (zero-variance / constant signals) rendered dark grey.
    Figure size scales with N: cell_px=52 px per cell keeps labels readable.
    Asset group boundaries drawn as white dividers.
    Practical for N < ~30; for larger portfolios, use plot_correlation_pairs instead.
    """
    corr, alpha_sorted, asset_map, labels, _ = _build_corr_matrix(portfolio_dir)
    N = len(alpha_sorted)

    cmap = plt.cm.RdBu_r.copy()
    cmap.set_bad(color="#2a2a2a")  # NaN → dark grey

    cell_px = 52
    fig_in = (N * cell_px + 280) / 100
    fig, ax = plt.subplots(figsize=(fig_in, fig_in))

    im = ax.imshow(corr, cmap=cmap, vmin=-1, vmax=1)

    for i in range(N):
        for j in range(N):
            v = corr[i, j]
            if np.isnan(v):
                continue
            weight = "bold" if abs(v) > 0.4 and i != j else "normal"
            txt_color = "white" if abs(v) > 0.65 else "black"
            ax.text(j, i, f"{v:.2f}", ha="center", va="center",
                    fontsize=6, color=txt_color, fontweight=weight)

    ax.set_xticks(range(N))
    ax.set_xticklabels(labels, rotation=45, ha="right", fontsize=7)
    ax.set_yticks(range(N))
    ax.set_yticklabels(labels, fontsize=7)

    asset_order = [asset_map.get(a) for a in alpha_sorted]
    for i in range(1, N):
        if asset_order[i] != asset_order[i - 1]:
            ax.axhline(i - 0.5, color="white", linewidth=2)
            ax.axvline(i - 0.5, color="white", linewidth=2)

    cbar = fig.colorbar(im, ax=ax, fraction=0.03, pad=0.02)
    cbar.set_label("Pearson ρ", fontsize=9)
    ax.set_title("Correlation Matrix (full-sample, daily signals)", fontsize=11, pad=10)

    plt.tight_layout()
    plt.savefig(out, dpi=100, bbox_inches="tight")
    plt.close()


def plot_correlation_pairs(portfolio_dir: Path, out: Path):
    """Horizontal bar chart of top 25 alpha pairs sorted by |ρ| descending.

    Bars coloured by ρ (RdBu_r). Pairs with undefined correlation (NaN) excluded.
    Scales to any N — readable even at 100 alphas where the matrix breaks down.
    """
    corr, alpha_sorted, asset_map, labels, _ = _build_corr_matrix(portfolio_dir)
    N = len(alpha_sorted)

    pairs = []
    for i in range(N):
        for j in range(i + 1, N):
            v = corr[i, j]
            if np.isnan(v):
                continue
            pairs.append((labels[i], labels[j],
                          asset_map.get(alpha_sorted[i]), asset_map.get(alpha_sorted[j]),
                          v))
    pairs.sort(key=lambda x: abs(x[4]), reverse=True)
    pairs = pairs[:25]

    if not pairs:
        return

    n_pairs = len(pairs)
    pair_labels = [f"{a}  ×  {b}" for a, b, _, _, _ in pairs]
    rho_vals = np.array([v for *_, v in pairs])

    cmap = plt.cm.RdBu_r
    norm = plt.Normalize(-1, 1)
    colors = [cmap(norm(v)) for v in rho_vals]

    row_h = 0.28
    fig_h = max(4, n_pairs * row_h + 1.5)
    fig, ax = plt.subplots(figsize=(12, fig_h))

    y = np.arange(n_pairs)
    bars = ax.barh(y, rho_vals, color=colors, height=0.7, edgecolor="none")

    # ρ value label: inside bar if wide enough, outside otherwise
    for bar, v in zip(bars, rho_vals):
        bar_w = abs(bar.get_width())
        inside = bar_w > 0.15
        if inside:
            x_pos = v / 2
            ha, txt_color = "center", "black"
        else:
            x_pos = v + (0.02 if v >= 0 else -0.02)
            ha, txt_color = ("left" if v >= 0 else "right"), "black"
        ax.text(x_pos, bar.get_y() + bar.get_height() / 2,
                f"{v:.3f}", va="center", ha=ha, fontsize=8, color=txt_color, fontweight="bold")

    ax.set_yticks(y)
    ax.set_yticklabels(pair_labels, fontsize=6.5)
    ax.invert_yaxis()
    ax.axvline(0, color="gray", linewidth=0.6)
    ax.set_xlim(-1.15, 1.15)
    ax.set_xlabel("Pearson ρ")
    ax.set_title(f"Top {n_pairs} Signal Correlation Pairs by |ρ|  (full-sample, daily)", fontsize=11)
    ax.grid(True, axis="x", alpha=0.2)

    sm = plt.cm.ScalarMappable(cmap=cmap, norm=norm)
    sm.set_array([])
    fig.colorbar(sm, ax=ax, fraction=0.015, pad=0.01, label="ρ")

    plt.tight_layout()
    plt.savefig(out, dpi=120, bbox_inches="tight")
    plt.close()

def _load_exposures(portfolio_dir: Path):
    """Shared loader: returns (times, exposures_dict, total, assets, asset_colors, n, alpha_ids, asset_map)."""
    signals = pl.read_parquet(portfolio_dir / "signals.parquet").sort("start_time")
    meta = pl.read_parquet(portfolio_dir / "metadata.parquet")

    alpha_ids = [c for c in signals.columns if c != "start_time"]
    n = len(alpha_ids)
    weight = 1.0 / n

    asset_map = {row["custom_id"]: row["base_asset"] for row in meta.iter_rows(named=True)}
    assets = sorted(set(asset_map.values()))
    times = signals["start_time"].to_list()

    exposures: dict[str, np.ndarray] = {}
    for asset in assets:
        asset_alphas = [aid for aid in alpha_ids if asset_map.get(aid) == asset]
        arr = np.array([signals[aid].fill_null(0).to_numpy() for aid in asset_alphas])
        exposures[asset] = arr.sum(axis=0) * weight

    total = sum(exposures[a] for a in assets)
    asset_colors = {"BTC": "#F7931A", "ETH": "#627EEA"}
    return times, exposures, total, assets, asset_colors, n, alpha_ids, asset_map


def _draw_gradient_exposure(ax, times, e_pct: np.ndarray, color: str):
    """Fill area + gradient line (RdYlGn mapped to value) for one exposure series."""
    ax.fill_between(times, 0, e_pct, where=e_pct >= 0, color=color, alpha=0.20)
    ax.fill_between(times, e_pct, 0, where=e_pct < 0, color=color, alpha=0.10)

    x_num = mdates.date2num(times)
    points = np.array([x_num, e_pct]).T.reshape(-1, 1, 2)
    segments = np.concatenate([points[:-1], points[1:]], axis=1)
    norm = TwoSlopeNorm(vmin=e_pct.min() if e_pct.min() < 0 else -1e-9,
                        vcenter=0,
                        vmax=e_pct.max() if e_pct.max() > 0 else 1e-9)
    lc = LineCollection(segments, cmap="RdYlGn", norm=norm, linewidth=1.2, zorder=3)
    lc.set_array(e_pct[:-1])
    ax.add_collection(lc)
    return lc


def _annotate_exposure_stats(ax, e_pct: np.ndarray, y_lim: float):
    """Dashed lines at min/mean/max on right edge; mean+median text box top-left."""
    mn, mx = e_pct.min(), e_pct.max()
    mean, med = e_pct.mean(), np.median(e_pct)

    for val, label, color in [
        (mn,   f"min {mn:.1f}%",    "#e74c3c"),
        (mean, f"mean {mean:.1f}%", "#f39c12"),
        (mx,   f"max {mx:.1f}%",   "#27ae60"),
    ]:
        ax.axhline(val, color=color, linewidth=0.7, linestyle="--", alpha=0.6, zorder=2)
        ax.text(0.995, (val - (-y_lim)) / (2 * y_lim),
                label, transform=ax.transAxes,
                fontsize=7, color=color, ha="right", va="center", alpha=0.9)

    ax.text(0.01, 0.97,
            f"mean {mean:.1f}%  |  median {med:.1f}%",
            transform=ax.transAxes, fontsize=7.5,
            color="white", alpha=0.85, va="top",
            bbox=dict(boxstyle="round,pad=0.2", facecolor="#1a1a1a", alpha=0.6, edgecolor="none"))

def _daily_returns(df: pl.DataFrame, prices: pl.DataFrame, market_col: str):
    """Aggregate portfolio PnL and market price to daily returns.

    Hourly beta ≈ 0 (signals hold for hours/days, not ticking every bar).
    Daily aggregation reveals the true regime correlation (~0.15-0.20 beta,
    ~0.75 correlation for this BTC/ETH portfolio).
    """
    combined = (
        df.join(prices.select("start_time", market_col), on="start_time", how="inner")
        .sort("start_time")
        .group_by_dynamic("start_time", every="1d")
        .agg(
            pl.col("pnl").sum().alias("port_ret"),
            pl.col(market_col).last().alias("mkt_close"),
        )
        .sort("start_time")
        .with_columns(pl.col("mkt_close").pct_change().alias("mkt_ret"))
        .drop_nulls()
    )
    return (
        combined["start_time"].to_list(),
        combined["port_ret"].to_numpy(),
        combined["mkt_ret"].to_numpy(),
    )


def plot_market_beta(portfolio_dir: Path, df: pl.DataFrame, out: Path):
    """Scatter of daily portfolio returns vs market, with full-sample OLS regression line.

    Beta is computed on daily returns (hourly beta ≈ 0 due to signal lag).
    Annotates β, R², and Pearson correlation.
    """
    prices = pl.read_parquet(portfolio_dir / "prices.parquet").sort("start_time")
    asset_cols = [c for c in prices.columns if c != "start_time"]
    market_col = "BTC" if "BTC" in asset_cols else asset_cols[0]

    _, port_ret, mkt_ret = _daily_returns(df, prices, market_col)

    beta = np.cov(port_ret, mkt_ret)[0, 1] / np.var(mkt_ret)
    alpha = np.mean(port_ret) - beta * np.mean(mkt_ret)
    corr = np.corrcoef(port_ret, mkt_ret)[0, 1]
    r_sq = corr ** 2

    x_line = np.linspace(mkt_ret.min(), mkt_ret.max(), 200)
    y_line = alpha + beta * x_line

    fig, ax = plt.subplots(figsize=(8, 7))
    ax.scatter(mkt_ret, port_ret, s=8, alpha=0.4, color="#2c3e50", label="Daily returns")
    ax.plot(x_line, y_line, color="#e74c3c", linewidth=1.5,
            label=f"OLS: β={beta:.3f}, α={alpha:.5f}")
    ax.axhline(y=0, color="gray", linewidth=0.4, alpha=0.5)
    ax.axvline(x=0, color="gray", linewidth=0.4, alpha=0.5)
    ax.set_title(f"Portfolio vs {market_col} Daily Returns\nR²={r_sq:.3f}, ρ={corr:.3f}")
    ax.set_xlabel(f"{market_col} Daily Return")
    ax.set_ylabel("Portfolio Daily Return")
    ax.legend(fontsize=9)
    ax.grid(True, alpha=0.2)
    plt.tight_layout()
    plt.savefig(out, dpi=150)
    plt.close()


def plot_alpha_vs_market(portfolio_dir: Path, df: pl.DataFrame, out: Path):
    """CAPM decomposition using full-sample beta: portfolio = Jensen's α + β × market.

    Uses a single full-sample OLS beta (not rolling) for a clean, stable
    decomposition of cumulative returns:
      - Portfolio: raw cumulative daily return
      - β × Market: market-explained portion (constant full-sample β)
      - Jensen's α: portfolio − β × market (pure excess return)
    """
    prices = pl.read_parquet(portfolio_dir / "prices.parquet").sort("start_time")
    asset_cols = [c for c in prices.columns if c != "start_time"]
    market_col = "BTC" if "BTC" in asset_cols else asset_cols[0]

    times, port_ret, mkt_ret = _daily_returns(df, prices, market_col)

    # Full-sample OLS beta
    beta = np.cov(port_ret, mkt_ret)[0, 1] / np.var(mkt_ret)

    port_equity = np.cumsum(port_ret)
    beta_equity = beta * np.cumsum(mkt_ret)
    alpha_equity = port_equity - beta_equity

    fig, ax = plt.subplots(figsize=(14, 5))
    ax.plot(times, port_equity, linewidth=0.8, color="#c0392b", label="Portfolio (total return)")
    ax.plot(times, beta_equity, linewidth=0.8, color="#7f8c8d",
            label=f"β×{market_col} (market-explained, β={beta:.3f})", alpha=0.8)
    ax.plot(times, alpha_equity, linewidth=1.2, color="#27ae60",
            label="Jensen's α (excess return)", linestyle="--")
    ax.axhline(y=0, color="gray", linewidth=0.4, alpha=0.5)
    ax.set_title(f"CAPM Decomposition: Portfolio = Jensen's α + β×{market_col}  (β={beta:.3f})")
    ax.set_ylabel("Cumulative Return (Arithmetic)")
    ax.legend()
    ax.grid(True, alpha=0.2)
    ax.xaxis.set_major_formatter(mdates.DateFormatter("%Y-%m"))
    ax.xaxis.set_major_locator(mdates.MonthLocator(interval=4))
    fig.autofmt_xdate()
    plt.tight_layout()
    plt.savefig(out, dpi=150)
    plt.close()
```
